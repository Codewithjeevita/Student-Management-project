🎓 Student Management System-project

A simple and efficient Flask-based web application to manage student records.


🚀 Features
- 🔐 Admin Login
- ➕ Add Students
- 📋 View Records
- ✏️ Edit & Delete
- 🔍 Search & Filter
- 📤 Export to CSV

🛠️ Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

▶️ Run Project
pip install flask
python app.py

Open: http://127.0.0.1:5000/
